
ULTRA WebAR Navigation - Quick Start

1. Extract the ZIP.
2. Open the folder with VS Code.
3. Run a local server (example: Live Server extension).
4. Open the site on your phone browser.
5. Allow camera access.
6. Point the camera at marker.png.

To edit the navigation path:
Open /map/path.json and modify the coordinates.

Example:
{"x":1,"z":-2}

Each point places a glowing arrow in AR space.
