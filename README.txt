Hospital WebAR Navigation (PRO Version)

HOW TO USE:
1. Upload this folder to GitHub Pages or any hosting.
2. Open the link on your mobile browser.
3. Point camera to HIRO marker.
4. Select destination to see navigation.

EDIT:
- Open app.js to change routes
- Open index.html to add more UI or rooms

RECOMMENDED:
- Print HIRO marker
- Place markers per floor
